Quick Start

Generate a dataset in two lines of Python or one CLI command.

Python 
from dupehell import generate
generate(
    domain="publishing",
    size=10000
)
CLI 
$ cargo run --release -- \
    --domain kyc \
    --size 100000

Features

What makes DupeHell the most comprehensive synthetic ER benchmark generator.

Noise Engine

8 categories, 35+ noise types — typos, OCR errors, homoglyphs, date swaps, phonetic variants, Unicode pollution.

Hard Negatives

8 adversarial patterns — same_field, mix_identifier, same_email, mix_arrays, and more — for robust evaluation.

Watermarking

3-layer SHA256 fingerprinting — metadata, canary records, numeric steganography across 10 columns.

40 Domains

KYC, healthcare, ecommerce, gaming, fintech, publishing, and 34 more — each with a purpose-built schema.

Dual Format

Arrow IPC for maximum throughput or Parquet ZSTD(3) for compact storage. Same data, identical F1.

Deterministic

Same seed + domain + difficulty = byte-identical output every time. Perfect for reproducible research.
